# Screenshot Shot List

## Suggested Screenshots

1. `首页 / 决策中心`
   Show the overview cards and the left-side scenario list.

2. `超大仓位拦截 / 回放详情`
   Show replay steps, decision summary, and blocked action.

3. `风险审计`
   Show policy hits for oversized position and leverage.

4. `事故报告`
   Show the generated incident report panel.

5. `告警适配器`
   Show console / webhook / Telegram previews.

6. `分析页`
   Show anomaly insights and baseline metrics.

## Capture Tips

- Prefer one blocked scene and one warning scene
- Keep screenshots focused; avoid showing too many collapsed sections
- If possible, use the Chinese UI consistently
- Name files in a simple order:
  - `01-home.png`
  - `02-replay-blocked.png`
  - `03-risk-audit.png`
  - `04-incident-report.png`
  - `05-alert-adapters.png`
  - `06-analytics.png`
